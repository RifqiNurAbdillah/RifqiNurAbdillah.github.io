Web Jual Beli Biji Kopi

Web ini menggunakan database MySQL dan backend Golang dengan JavaScript pada frontend.

Penjelasan Web

Web frontend bisa diakses di sini.
Backend belum di-host, jika ingin mencoba backend silakan clone repository di sini.

Petunjuk menjalankan backend:

  • Clone/download repository.
  • Nyalakan XAMPP dan buat database tokokopi atau ubah dsn di Golang: dsn := "root@tcp(localhost:3306)/tokokopi?charset=utf8mb4&parseTime=True&loc=Local"
  • Ubah nama database sesuai keinginan.
