Tentang Saya

Saya Mahasiswa Semester 5 Teknik Informatika Universitas PGRI Semarang dengan fokus pada Web Development dan Internet of Things (IoT). Memiliki kemampuan dalam pengembangan aplikasi berbasis web menggunakan Go (Golang) untuk backend, serta memahami integrasi dengan MySQL dan API sederhana. Berpengalaman dalam membuat sistem otomatisasi berbasis sensor dengan ESP8266 serta mengimplementasikan konsep logika pemrograman, struktur data, dan komputasi cerdas dalam proyek nyata.

Melalui website ini, saya bermaksud menampilkan berbagai hasil proyek yang telah saya lakukan. Website ini saya gunakan sebagai media untuk menyajikan informasi, dokumentasi dari setiap proyek yang saya kerjakan.
